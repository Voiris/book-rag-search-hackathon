# 📚 Book Search RAG
### Book Search RAG - это локальный RAG (Retrieval-Augmented Generation) сервис по книгам в формате .txt.
#### Приложение умеет отвечать на вопросы, используя информацию из загруженных книг. Оно работает на основе модели LLaMA 2 7B и использует библиотеку LangChain для обработки запросов и генерации ответов.

## Запуск приложения (в ручную)
### Требования:
- Python 3.10+
- pip
- Ollama (+модель Gemma3)

### Инструкции по запуску:
1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Voiris/book-rag-search-hackathon
    cd book-rag-search-hackathon
    ```

2. Установите зависимости
    ```bash
    pip install -r requirements.txt
    ```

3. Установка Ollama:
   1. Установщик Ollama можно найти на официальном сайте: https://ollama.com/download
   2. После установки Ollama, загрузите модель Gemma3:
      ```bash
      ollama pull gemma3
      ```
   3. Убедитесь, что Ollama работает корректно, запустив команду:
      ```bash
      ollama list
      ```

4. Запустите приложение (выберите интерфейс):
    #### В Tkinter:
    ```bash
    python main.py
    ```
    #### В Streamlit (экспериментальный):
    ```bash
    streamlit run main.py
    ```

### Использование:
1. В интерфейсе приложения есть поле для запроса и 4 кнопки. Для начала вам надо загрузить минимум одну книгу в формате .txt, нажав на кнопку "Добавить книгу". Выберите файл книги на вашем компьютере. (например, example_books/metro2033.txt)
2. При нажатии кнопки "Книги" вы увидите список загруженных книг.
3. В поле для запроса введите вопрос, на который вы хотите получить ответ, используя информацию из загруженных книг.
4. Нажмите кнопку "Ответить", и приложение сгенерирует ответ на основе информации из книг и AI модели. (ответ может занять некоторое время, в зависимости от сложности вопроса и объема информации в книгах)
5. Кнопкой поиск вы можете найти цитаты, в которых встречаются ключевые слова из вашего запроса. Это может помочь вам найти конкретные места в книгах, которые могут быть полезны для ответа на ваш вопрос.

### Конфигурация:
- Вы можете настроить параметры модели и RAG в файле `config.py`. Например: выбрать AI модель, изменить количество возвращаемых фрагментов, размер chuck'ов и другие параметры, чтобы оптимизировать производительность и качество ответов.
